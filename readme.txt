== Description ==
Convert your Telegram channels to Wordpress website.

This Wordpress plugin can extract posts content from different Telegram channels.

Auto publish many post daily from Telegram channels and get visitor from Search Engines.
TelePost doesn't need to add any bot to the selected Telegram channels and it can crawl any public Telegram channels.

How it's work:

After installing the plugin you will see "TelePost Settings" item in your dashbaord menu.
In the settings page enter your desired Telegram channels username.
Channel username must be entered without @ and https://t.me
Also you can specify the crawl schedule rate.

Warning : This version doesn't support fetching video post from Telegram.
